Install
-------------------------------------------------------------------------------

``pyclopedia`` is released on PyPI, so all you need is:

.. code-block:: console

	$ pip install pyclopedia

To upgrade to latest version:

.. code-block:: console

	$ pip install --upgrade pyclopedia
